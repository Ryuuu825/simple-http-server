Features

  • Static file server with live reload
  • Reverse proxy (path-based and port-based)
  • Web-based admin panel
  • Auto-detect local IP for network access
  • File download support
  • Recursive directory watching
  • Automatic port assignment

Usage

macOS / Linux:

chmod +x simple-http-server-*
./simple-http-server-darwin-arm64

Windows:

simple-http-server-windows-amd64.exe
Note: The server will start on a random available port and automatically open the admin panel in your browser.
